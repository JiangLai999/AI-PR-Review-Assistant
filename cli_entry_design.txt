请实现 AI PR Review 助手的 CLI Entry 模块。

## 模块职责

1. CLI 入口（使用 click 框架）
2. 参数解析（PR URL、模型、输出格式等）
3. 流程编排（串联所有模块）
4. 终端美化输出（使用 Rich 库）
5. 错误处理和退出码

## 核心命令

```bash
# 基本用法
pr-review https://github.com/owner/repo/pull/123

# 指定模型
pr-review https://github.com/owner/repo/pull/123 --model claude-sonnet-4-20250514

# 导出 Markdown 报告
pr-review https://github.com/owner/repo/pull/123 --output report.md

# 发布为 GitHub PR Comment
pr-review https://github.com/owner/repo/pull/123 --publish-comment

# 显示详细信息
pr-review https://github.com/owner/repo/pull/123 --verbose
```

## 流程编排

```
1. 解析 CLI 参数
2. 加载配置
3. 初始化各模块：
   - PRFetcher
   - FilterPipeline
   - ContextBuilder
   - PromptAssembler
   - AIClient
   - PostProcessor
4. 执行 Review 流程：
   a. 获取 PR 数据
   b. 过滤文件
   c. 构建上下文
   d. 组装 Prompt
   e. 调用 AI 分析
   f. 后处理结果
5. 输出报告
6. 返回退出码
```
