请设计和实现 AI PR Review 助手的多模型供应商支持系统。

## 背景

当前项目只支持 Anthropic Claude 模型，需要扩展支持：
1. 官方 API（Anthropic、OpenAI、DeepSeek、阿里通义千问等）
2. 第三方中转站（API2D、CloseAI、OhMyGPT、OpenRouter 等）
3. 自定义 API 端点

## 设计要求

### 1. 统一的模型接口

```python
class BaseModelProvider(ABC):
    @abstractmethod
    async def chat(self, messages: list[dict], **kwargs) -> str:
        pass

    @abstractmethod
    def estimate_cost(self, input_tokens: int, output_tokens: int) -> float:
        pass
```

### 2. 支持的供应商

**官方 API：**
- Anthropic Claude
- OpenAI GPT
- DeepSeek
- 阿里通义千问
- 百度文心一言
- 讯飞星火
- 智谱 ChatGLM

**第三方中转站：**
- OpenRouter
- API2D
- CloseAI
- OhMyGPT
- 自定义端点

### 3. 配置结构

```json
{
  "model_provider": {
    "name": "deepseek",
    "display_name": "DeepSeek",
    "api_key": "sk-xxx",
    "base_url": "https://api.deepseek.com/v1",
    "model_name": "deepseek-chat",
    "api_format": "openai",
    "headers": {},
    "extra_params": {}
  }
}
```

### 4. 配置向导

实现交互式配置向导：
- pr-review config
- pr-review config --quick
- pr-review config --advanced
- pr-review config show

### 5. API 格式支持

- OpenAI 格式（兼容大多数中转站）
