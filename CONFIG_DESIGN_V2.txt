请参考 Claude Code 和 OpenCode 的配置方式，设计一个更好的 AI PR Review 助手配置方案。

## 参考配置

### Claude Code 配置特点
- 使用环境变量方式配置
- 支持自定义 API 端点
- 支持多模型配置
- 简洁明了

### OpenCode 配置特点
- 使用 JSON 配置文件
- 支持多供应商
- 每个供应商有独立的配置
- 支持模型参数配置
- 结构清晰

## 设计要求

1. 结合两者的优点
2. 支持多供应商配置
3. 支持自定义 API 端点
4. 支持模型参数配置
5. 配置界面美观易用
6. 支持配置导入导出
