请设计 AI PR Review 助手的图形界面配置方案。

## 需求

用户希望有图形界面来配置模型供应商，而不是纯命令行交互。

## 可选方案

### 方案 1：Web 界面配置
- 使用 Flask/FastAPI 创建本地 Web 服务器
- 在浏览器中打开配置页面
- 可视化选择供应商、输入 API Key
- 支持配置验证

### 方案 2：桌面应用配置
- 使用 PyQt/Tkinter 创建桌面配置工具
- 独立的配置应用程序
- 原生界面体验

### 方案 3：浏览器配置
- 启动本地服务器
- 自动打开浏览器
- 配置完成后关闭服务器

## 推荐方案

推荐方案 1 或方案 3，因为：
1. 开发成本低
2. 跨平台支持
3. 界面美观
4. 易于维护

## 需要创建的功能

1. 可视化选择模型供应商
2. 输入 API Key（带验证）
3. 选择模型名称
4. 配置预览
5. 一键测试配置
6. 保存配置
7. 导入/导出配置

## 界面设计要求

1. 简洁美观
2. 响应式设计
3. 中文界面
4. 友好的错误提示
5. 配置验证反馈

请设计最佳的图形界面配置方案。
