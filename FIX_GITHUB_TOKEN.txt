### 2. 添加获取说明

```
获取 GitHub Token：

1. 访问 https://github.com/settings/tokens
2. 点击 Generate new token → Generate new token (classic)
3. 填写名称（如 "AI PR Review"）
4. 选择权限：repo（完整仓库访问）
5. 点击 Generate token
6. 复制生成的 Token

Token 格式：ghp_xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
```

### 3. 验证 Token 格式

```python
def validate_github_token(token: str) -> bool:
    """验证 GitHub Token 格式"""
    if not token:
        return False
    if not token.startswith("ghp_"):
        return False
    if len(token) < 40:
        return False
    return True
```

### 4. 必填验证

```python
while True:

    if validate_github_token(github_token):
```
