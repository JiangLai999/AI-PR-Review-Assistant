# Asset Guide

## Icon Directory

Final project icons should be stored in:

```text
assets/icons/
```

See:

```text
assets/icons/README.md
```

for the naming convention and recommended export sizes.

## Minimum Final Asset Set

- `icon-main.png`
- `icon-repo-avatar.png`
- `icon-app.png`
- `icon-transparent.png`
- `icon-hero-glow.png`

## Usage Summary

- `icon-main.png`: default project branding
- `icon-repo-avatar.png`: GitHub avatar / square preview
- `icon-app.png`: app launcher / installer icon source
- `icon-transparent.png`: flexible document / slide usage
- `icon-hero-glow.png`: demo cover / presentation artwork
