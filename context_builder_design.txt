## 数据模型

```python
class FileContext(BaseModel):
    file_path: str
    language: str
    diff: str
    diff_with_context: str
    imports: list[str]
    functions: list[FunctionInfo]
    classes: list[ClassInfo]
    parse_mode: str  # "tree_sitter" | "regex" | "fallback"

class FunctionInfo(BaseModel):
    name: str
    start_line: int
    end_line: int
    parameters: list[str]
    return_type: str | None
    is_async: bool

class ClassInfo(BaseModel):
    name: str
    start_line: int
    end_line: int
    methods: list[str]
    parent_classes: list[str]
```
