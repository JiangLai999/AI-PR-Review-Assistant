# Icon Assets

This directory stores the final icon set for `AI-PR-Review-Assistant`.

## Recommended Files

- `icon-main.png`
: Primary project icon for README, release pages, and general branding.
- `icon-repo-avatar.png`
: Simplified square avatar for the GitHub repository or profile-style display.
- `icon-app.png`
: App-style icon source for Windows/macOS packaging.
- `icon-transparent.png`
: Transparent background version for slides, docs, or mixed backgrounds.
- `icon-hero-glow.png`
: Presentation / marketing version with stronger visual emphasis.

## Recommended Sizes

- Main icon: `1024x1024`
- Repo avatar: `512x512`
- App icon source: `1024x1024`
- Transparent version: `1024x1024`
- Hero / presentation image: `1600x900` or `1920x1080`

## Suggested Mapping

- Main icon: flat GitHub-style version
- Repo avatar: monogram / initials version
- App icon: platform-style rounded icon version
- Transparent backup: transparent background version
- Hero artwork: glowing / presentation version

## Notes

- Keep exported source files lossless where possible.
- Prefer PNG for raster assets.
- If an SVG source is available, store it alongside the PNG exports.
- Avoid committing temporary drafts; only keep the approved final assets here.
